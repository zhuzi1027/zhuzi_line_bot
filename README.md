## 程式內容
  - base
    
    基礎程式內容指使用app.py作為開發，並將檔案儲存為Excel表
  - Advamced
    - base : 加入json檔用以存放重要資訊(txt, ini檔皆可)
    - advamced : 將程式內的函數單獨分離出來而外製作一個封包存放這接函數
    - to_json : 將儲存在Excel表改為儲存在json檔中
    - to_database : 將儲存的資料儲存在資料庫中
